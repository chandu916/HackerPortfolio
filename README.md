# Chandu Portfolio

Premium single-page portfolio for a Cyber Security Engineer, built with React, TypeScript, and Vite.

## Highlights

- Scroll-snap, full-screen section system
- Cinematic hero with low-opacity binary rain and terminal-inspired motion
- Modular sections for about, skills, projects, team, and contact
- Glassmorphism panels, neon borders, and controlled hover interactions
- Custom cursor, magnetic buttons, tilt cards, and accessible project modal

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
